# Idea
Una curva puede ser una suma de otras curvas, y si esas son suaves, entonces la resultante es seccionalmente suave. 

# Definición
Una curva $\Gamma$ se dice **seccionalmente suave** si $\Gamma=\Gamma_{1}+\Gamma_{2}+\dots+\Gamma_{n}$, donde $\Gamma_{i}$ son suaves. O sea una curva es seccionalmente suave si se puede escribir como unión de curvas suaves.
