# Definiciones Iniciales
## Algoritmo
Un algoritmo es una secuencia lógica, finita y ordenada de pasos que permiten resolver un problema o ejecutar una tarea.
Son un conjunto de instrucciones que se deben realizar para lograr un resultado.
*ej: Receta de cocina.*

## Programa
Un programa es una representación computacional de un algoritmo, escrito en un determinado lenguaje de programación.

- El lenguaje de programación (Python) es capaz de traducirse a un código binario (de ceros y unos) que el computador puede entender.

# Tipos de Datos

| Tipo de dato | Descripción                       | Ejemplo        |
| ------------ | --------------------------------- | -------------- |
| int          | Números **enteros** (sin decimal) | 1, -7, 0       |
| float        | Números reales (con decimales)    | 1.0, -9.2      |
| str          | Cadenas de texto                  | "Hola", 'Hola' |
|              |                                   |                |
## Convertir entre tipos de datos
Funciones para convertir entre tipos de datos:
\*poner dato a convertir dentro de paréntesis.

1. Convertir a int: `int()`
2. Convertir a float: `float()`
3. Convertir a string: `str()`

*ejemplo:*
```py
int(2.7)
```

```py
float(7)
```

```py
str(11)
```

```py
str(2.7)
```

también se pueden convertir cadenas de texto a números:
```py
>>> int('70')
70


>>> float('2.5')
2.5
```

Y se debe notar el tipo de dato que se busca almacenar:
```py
int('2.5')
```

```py
float('perro')
```

## Expressions (Expresiones)
Las expresiones son valores que interactúan entre sí usando **operadores**. Producen un nuevo valor al evaluarse:
```py
2+3
```

```py
5*7
```

```py
'hola'+' '+'mundo'
```

```py
7>5
```

En cada una de las expresiones, python evalúa los valores y operadores para generar un nuevo valor como resultado.

**Operador positivo**: convierte el valor entregado a positivo
