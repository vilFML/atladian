# Experimento Aleatorio
Un experimento aleatorio es un experimento cuyo resultado solo se conoce después de realizarlo

*Ej:*
1. $E_{1}$: lanzar una moneda
2. $E_{2}$: Lanzar $n$ monedas
3. $E_{3}$: Medir la duración de una ampolleta

# Espacio Muestral
Dado un experimento aleatorio $E$, se define el *espacio muestral* $\Omega$ asociado a $E$ como **el conjunto de resultados posibles**.

*Ej:*
1. $E_{1}$: lanzar una moneda; $\Omega_{1}=\{ \text{cara, sello} \} = \{ C,S \}$
2. $E_{2}$: Lanzar $n$ monedas; $\Omega_{2}=\{ (x_{1},x_{2},x_{3},\dots,x_{n}):x_{i}\in \{ C,S \} \}$
3. $E_{3}$: Medir la duración de una ampolleta; $\Omega_{3}=\{ x:x\geq 0 \}=\mathbb{R}^{+}$

 **Obs 1:** El espacio muestral no es necesariamente un conjunto numérico
**Obs 2:** Dependiendo de la cardinalidad del espacio muestral $|\Omega|$, se tienen los casos:
1. $\Omega$ finito, *Ej*: Experimento de lanzar una moneda.
2. $\Omega$ infinito numerable. *Ej* experimento de lanzar $n$ monedas.
3. $\Omega$ infinito no numerable. *Ej* medir duración de ampolleta.
